Python 简易文本字数统计工具
 
项目介绍
 
这是一个零基础友好的 Python 小工具，核心功能是快速统计文本中的中文字数、英文字数、总字符数（含标点/空格），支持直接输入文本或读取 TXT 文件统计，无需复杂配置，新手也能直接运行使用。
 
技术栈
 
- 核心语言：Python 3.6+
- 依赖工具：无额外依赖（仅使用 Python 内置库）
 
安装与运行
 
1. 下载项目：打开仓库链接→Code→Download ZIP，解压到本地；
2. 准备环境：安装 Python 3.6 及以上版本（官网 https://www.python.org/ 可下载）；
3. 安装依赖：无（无需额外安装库）；
4. 运行项目：双击运行  main.py  文件，或打开命令行进入项目文件夹，输入  python main.py  即可启动。
 
核心功能
 
1. 支持两种统计模式：直接输入文本统计 + 读取本地 TXT 文件统计；
2. 自动区分中文字符、英文字符，分别输出统计结果，同时显示总字符数；
3. 统计速度快，支持任意长度文本（TXT 文件无大小限制）。
